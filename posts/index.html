Local Website on RPi4 with Hugo and Jenkins

Introduction How to install Jenkins on windows: https://www.jenkins.io/doc/book/installing/windows/ Hugo: https://gohugo.io/getting-started/quick-start/ installed using choko set nginx on RPi4 to server hugo: https://gideonwolfe.com/posts/sysadmin/hugonginx/ Conversation with ChatGPT: https://chatgpt.com/share/0ca4052b-164a-4020-ab38-f74ac0907161 Deploying a static Hugo site with NGINX: https://gideonwolfe.com/posts/sysadmin/hugonginx/ Assuming that the development location on my windows machine is D:\website\hugo\my-blog and nginx is installed on target server (RPi4), the final Jenkins pipeline script: pipeline { agent any environment { RPI_CREDS = credentials('96877582-94e3-4fc4-93ed-e84e2c8be6d6') RPI_HOST = '10.0.0.169' RPI_WEBSITE_DIR = '/var/www/html' } stages { stage('Build') { steps { dir('D:\\website\\hugo\\my-blog') { // Build the Hugo site bat "hugo.exe" } } } stage('Deploy') { steps { dir('D:\\website\\hugo\\my-blog') { // Create an archive of the built site bat "powershell Compress-Archive -Force -Path .\\public\\* -DestinationPath site.zip" // bat """ plink -v -ssh -pw ${RPI_CREDS_PSW} ${RPI_CREDS_USR}@${RPI_HOST} "cd ${RPI_WEBSITE_DIR}; echo -e ${RPI_CREDS_PSW} | sudo -S rm -R *" """ // Copy the archive to the Raspberry Pi bat """ pscp -pw ${RPI_CREDS_PSW} site.zip ${RPI_CREDS_USR}@${RPI_HOST}:${RPI_WEBSITE_DIR}/site.zip """ // Connect to the RPi and extract the archive bat """ plink -v -ssh -pw ${RPI_CREDS_PSW} ${RPI_CREDS_USR}@${RPI_HOST} "echo -e ${RPI_CREDS_PSW} | sudo -S unzip -o ${RPI_WEBSITE_DIR}/site.zip -d ${RPI_WEBSITE_DIR}; echo -e ${RPI_CREDS_PSW} | sudo -S rm ${RPI_WEBSITE_DIR}/site.zip" """ } } } } } I build this pipeline on my local machine with built-in node.
